QRTrippin Data Deletion Policy

If you wish to request the deletion of your personal data associated with the QRTrippin Facebook app, please follow these steps:

Option 1: Delete Data via Facebook

You can request data deletion directly through Facebook by following these steps:

  1. Go to your Facebook App Settings.
  2. Find "QRTrippin" in the list of connected apps.
  3. Click "Remove" and choose the option to delete all data associated with this app.

Option 2: Request Data Deletion via Email

If you prefer, you can request manual data deletion by sending an email to: QRTrippin@gmail.com with the subject line: Data Deletion Request. Please include your Facebook ID for verification.

We will process your request within 7 business days.
